# Social Platform (Still ongoing development)
Fullstack Social Platform using **Typescript**, **React**, **NodeJS**, and **GraphQL**.

# Tech Stack

* Typescript (Language)
* React (Frontend library)
* Next.js (Server side rendering)
* ChakraUI (UI design)
* GraphQL & TypeGraphQL (API queries)
* URQL (GraphQL client)
* Node.js (API server)
* PostgreSQL (Database)
* MikroORM (ORM)
* Redis (Sessions Data)
